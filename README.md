### link venv for jupyter

``` shell
pip install ipykernel
pip install ipython
pip install notebook
python -m ipykernel install --user --name=usopp
jupyter kernelspec list
jupyter notebook  --allow-root

# uninstall venv
jupyter kernelspec uninstall usopp
```
